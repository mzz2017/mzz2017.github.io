引言：当天折腾到凌晨4点，得到喜讯之后还是阔以。

2018.1.29注：此法越来越难成功了。

本文可以帮你从零套得谷歌云和阿里云的300美金新手礼金，仅仅需要一点时间和必要的工具。

本方法成功时间：2018/01/05 （后确认是挂掉了，不知道原因，猜测是美国公民信息被别人使用过的原因吧）

  • 2018年1月12日注：本方法注册谷歌云败北告终，但注册阿里云国际版后，可以成功消费一笔。当你支付后，paypal会在几天后告知你payoneer的银行账户扣款失败，但交易已无法中断。意思是这笔钱就由paypal帮你付掉了，你的阿里云服务器不会因此吊销，但是你的paypal账户也会因此被封。这样做的话如果这笔bill比较大的时候我不知道会不会有妖妖灵把我带走所以我只搞了9美金一个月的服务器，到期再说吧。

开始吧！

了解需求

Google Cloud需求

我们准备连着注册谷歌账户并且开通Google Cloud，因此需要下面的几个准备：

  1. 一把梯子（最好是美国梯子）
  2. 一套美国公民信息
  3. 一个美国电话号码
  4. 一个非Chrome浏览器（Chrome也行，但个人觉得切换用户比较麻烦，推荐使用Firefox，proxy插件使用FoxyProxy）
  5. 一个未注册过谷歌账户的邮箱
  6. 一张VISA信用卡或一个美国银行账户

阿里云国际版需求

  1. 一套美国公民信息
  2. 一个可接收验证码的美国电话号码
  3. 一个未注册过阿里云的邮箱
  4. 一张非国内VISA信用卡或一个非国内PayPal账户

开始准备

一套美国公民信息

  • 好维持 打开后即可看到一套随机生成的公民信息，将其截图妥善保存，可能会多次用到。

一个美国电话号码

  • 注册Google Cloud的过程中没有要求用这个号码接收验证码什么的，但是以防万一，我们最好申请一个虚拟美国电话。
  • textnow 虚拟美国电话，可接打电话和接发短信。
    • 注册时Name填写随机生成的美国公民信息即可。
    • 注意！注册时该网站使用谷歌人机验证系统，但谷歌被墙了，所以国内网会出现注册按钮点不了的情况，因此请务必踩着梯子注册。

一个美国银行账户

  • Payoneer
    • 注册需要美国公民信息和美国电话号码。银行账户我选的是个人、美国、Bank of America，账号填411774400000 0000（把0随便改几个就好），路由号填123456780（随便填了一个能填的），这个绑卡验证很水的，一般能通过。
    • 等待一段时间收到邮件通过后，在左上角的Receive - Global Payment Service中可以看到你的美国银行账户信息。

进阶准备

Optional: 一张美国VISA信用卡

  • Payoneer注册成功后，右上方有关于VISA的信息，点击操作申请即可。目前我的提交还在审核，还没得到成功的消息。（方法失败了）

阿里需求: 一个美国PayPal账户

  1. Payoneer注册成功后，注册PayPal选择绑定Bank，填写Payoneer中的美国银行账户信息，被告知等待两个1美元以下的汇款账单。
  2. 等待1~2天，Payoneer会以邮件通知你收到汇款。登录PayPal继续验证Bank，填写两封汇款的金额，绑定成功。成功得到一个美国PayPal账户。

正式注册

阿里云国际版

  • 添加支付信息时，选择PayPal即可，获得300美金。

Google Cloud

  1. 使用Firefox浏览器踩着梯子，登录Google Cloud 注册新的谷歌账号，填写美国公民信息和美国电话号码。
  2. 继续，注册Google Cloud时选择美国当地银行账户，填写Payoneer中的美国银行账户信息，被告知等待两个1美元以下的汇款账单。
  3. 等待1~2天，Payoneer会以邮件通知你收到汇款。登录Google Cloud继续验证Bank，填写两封汇款的金额，绑定成功。至此获得300美金。（2018.1.12注：最后发现虽然得到300美金但是谷歌要求我进行身份验证，要求我上传居民证以及银行卡照片，所以失败告终）
